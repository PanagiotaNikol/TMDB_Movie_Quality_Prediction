# TMDB Movie Quality Prediction using Machine Learning

## Project Overview

This project applies Machine Learning techniques to predict movie quality using the TMDB 5000 Movies Dataset.

The problem is formulated as a binary classification task, where movies are classified as:

- Good (target = 1)
- Not Good (target = 0)

based on their average rating (`vote_average`).

The project compares the performance of two classification models:

- Logistic Regression
- Random Forest

---

## Dataset

The analysis is based on the TMDB 5000 Movies Dataset.

After data preprocessing and removal of missing values, approximately 4,800 movie records remained available for training and evaluation.

### Features Used

The following features were selected for model training:

- budget
- runtime
- popularity
- vote_count

### Target Variable

The target variable was created using the movie average rating:

- vote_average ≥ 7 → Good (1)
- vote_average < 7 → Not Good (0)

---

## Data Preprocessing

Before training the models:

- Missing values were removed using `dropna()`
- Relevant features were selected
- The target variable was created
- The dataset was split into training and testing sets using an 80%-20% ratio

---

## Models Implemented

### Logistic Regression

Logistic Regression was used as a baseline classification model due to its simplicity, fast training process, and interpretability.

### Random Forest

Random Forest was used to capture more complex relationships between the dataset features.

Model parameters:

- n_estimators = 200
- max_depth = 10

Feature Importance analysis was also performed using the Random Forest model.

---

## Evaluation Metrics

The models were evaluated using:

- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix
